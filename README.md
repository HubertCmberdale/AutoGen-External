AutoGen: External Logic Controller
AutoGen is an external execution utility designed for flow automation and data-layer assistance (overlays). The software core focuses on low-latency processing and background process control with minimal system overhead.

⚠️ CRITICAL REQUIREMENT
WINDOWED MODE ONLY: This utility requires Roblox to be in Windowed Mode (Borderless or Windowed). It will NOT function in Fullscreen mode, as the display scaling will fail to center and the logic engine will remain inactive.

Technical Specifications
Execution Logic: Dual-mode architecture supporting AUTO (fully automated processing) and MANUAL (active visual overlay assistance).

Pathfinding Engine: High-speed algorithm for real-time optimal route calculation based on current game state.

Input Mapping: Configurable hotkey system for state control (Start/Stop/Toggle) without losing focus on the primary window.

Environment UI: Optimized dark-themed GUI designed for distraction-free data monitoring.

Standalone Process: Executes as a completely external binary to ensure environment stability and process integrity.

Deployment & Setup
Ensure Roblox is NOT in Fullscreen mode.

Execute the AutoGen_External.exe binary.

Map input hotkeys via the configuration panel.

Initialize the required operational mode (AUTO/MANUAL).

Monitor data flow and states via the integrated terminal log output.

Note: This utility is developed for technical analysis and personal use only.
